Справка: что где делать

👦 Ученики

Добавить: «Добавить ученика» или раздел Ученики → «+».

Направления: в карточке ученика — вкладка «Направления» или при создании.

Абонемент: карточка ученика → направление → «Пополнить абонемент» (сумма и число занятий).

📅 Расписание и посещения

Слоты: Расписание → Неделя → Редактировать — день, время, преподаватель.

Индивидуальные: в слоте обязательно выбрать ученика.

Посещения: Сегодня — отметить «Был» и «Оплачено», сохранить.

Разовое / отработка: вкладка «Сегодня» → «+ Разовое / отработка».

🎫 Абонементы

Раздел Абонементы — все карточки с прогрессом занятий.

Зелёная полоса — остаток занятий. Клик по карточке ведёт к ученику.

👩‍🏫 Преподаватели

Карточка: Преподаватели → клик по ФИО.

Зарплата: вкладка «Зарплата» — фильтр периода и направлений, посещения × цена, 30%.

Направления: назначить на вкладке «Направления» у преподавателя.

{% if is_admin %}
💳 Оплаты

Внести: Оплаты → раскрыть «+ Добавить оплату».

Редактировать: «✎ Редактировать» в таблице, изменить суммы и сохранить.

Excel: кнопка «📥 Excel» — выгрузка по текущим фильтрам.

📊 Отчёты

Сводка: Отчёты — доходы, расходы, зарплаты, графики за месяц.

Выберите месяц и год вверху страницы.

{% endif %} {% if is_admin %}

Директор — роль «Директор»: оплаты, отчёты, настройки.

Ресепшн — ученики, расписание, абонементы. Без финансов.

Пароль задаётся при создании пользователя.

{% endif %}

Эти цены используются в расчёте зарплаты преподавателей.
